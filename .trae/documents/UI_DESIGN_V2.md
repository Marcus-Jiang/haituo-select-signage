# HaituoSelect-Signage UI 设计方案

> 基于 UI/UX Pro Max 设计系统搜索，结合「专业 B2B」+「日式极简」融合风格。

---

## 配色方案

| 用途 | 颜色 | 色值 | 说明 |
|---|---|---|---|
| **主色** | 深海军蓝 | `#0F172A` | 品牌色、导航栏、按钮主色 |
| **强调色** | 专业蓝 | `#0369A1` | 悬停状态、链接、高亮 |
| **浅强调色** | 天蓝 | `#0EA5E9` | 次要强调、hover 效果 |
| **背景色** | 浅灰白 | `#F8FAFC` | 页面背景 |
| **卡片背景** | 纯白 | `#FFFFFF` | 产品卡片、模态框 |
| **主文字** | 深蓝黑 | `#020617` | 标题、正文 |
| **次文字** | 石板灰 | `#475569` | 描述、辅助文字 |
| **边框色** | 浅灰 | `#E2E8F0` | 卡片边框、分割线 |

## 字体方案

| 用途 | 字体 | 权重 |
|---|---|---|
| **标题 / 导航** | Noto Serif JP / SC | 600, 700 |
| **正文 / 按钮** | Noto Sans JP / SC | 400, 500 |
| **产品名称** | Noto Sans JP / SC | 500 |

> Google Fonts 导入：`Noto+Sans+SC:wght@400;500;700&family=Noto+Serif+SC:wght@600;700`

## 效果规范

| 效果 | 值 | 说明 |
|---|---|---|
| **卡片圆角** | `rounded-xl` (12px) | 产品卡片 |
| **模态框圆角** | `rounded-2xl` (16px) | 详情/场景模态框 |
| **悬停过渡** | `0.25s ease` | 所有可交互元素 |
| **卡片悬停** | `translateY(-2px) + box-shadow` | 微妙上浮 |
| **模态框背景** | `bg-black/50` ~ `bg-black/70` | 半透明遮罩 |
| **留白策略** | 大量留白，间距 `gap-5` `p-6` | 日式极简 |

## 组件布局

### 导航栏
```
[Logo/标题]  [语言切换] [面包屑]
[一级标签] [二级标签（选中一级后显示）]
```

### 产品卡片
```
┌──────────────────┐
│                   │
│   产品图片         │  ← 可点击跳转场景
│                   │
├──────────────────┤
│ 产品名称           │
├──────────────────┤
│ [详情]  [使用场景]  │  ← 2个按钮
└──────────────────┘
```

### 详情模态框（左图右文）
```
┌──────────────────────────────────────┐
│  ┌──────────────┐  ┌──────────────┐  │
│  │              │  │              │  │
│  │   产品大图    │  │  Markdown    │  │
│  │              │  │  渲染内容     │  │
│  │              │  │              │  │
│  └──────────────┘  └──────────────┘  │
│         ← 50% →    ← 50% →          │
└──────────────────────────────────────┘
```
