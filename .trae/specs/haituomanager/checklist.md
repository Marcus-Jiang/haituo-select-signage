- [ ] manage.html 能正常打开，显示中文界面
- [ ] 页面能正确加载并显示产品列表
- [ ] 每个产品条目左侧显示图片，右侧显示完整信息

## 产品管理
- [ ] 点击产品条目能打开编辑模态框
- [ ] 编辑模态框显示所有可编辑字段（分类、品名、描述、图片）
- [ ] 保存编辑后，data.json 正确更新
- [ ] 点击新增产品按钮能打开新增模态框
- [ ] 新增产品后，data.json 包含新产品
- [ ] 新增产品的图片保存到正确路径，命名符合规则
- [ ] 每个产品条目有删除按钮
- [ ] 点击删除弹出二次确认对话框
- [ ] 确认删除后，产品从 data.json 移除
- [ ] 确认删除后，相关图片文件被删除

## 分类管理
- [ ] 分类管理面板能展开和收起
- [ ] 能新增一级分类
- [ ] 能新增二级分类（在一级分类下）
- [ ] 能新增三级分类（在二级分类下）
- [ ] 能编辑分类名称（中日双语）
- [ ] 能删除分类（无子分类/产品时）

## 图片处理

- [ ] 上传的图片按照现有文件夹结构保存

- [ ] 编辑时能替换图片
- [ ] 替换图片时，旧图片被删除，新图片正确保存
